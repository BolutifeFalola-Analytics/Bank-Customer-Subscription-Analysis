# Bank Customer Subscription Analysis

## Overview
This project analyzes customer data to identify factors influencing subscription to bank products. The goal is to generate insights that support data-driven decision-making in banking and marketing.

## Objectives
- Identify key drivers of customer subscription
- Analyze customer demographics and financial attributes
- Generate actionable business insights

## Dataset
- Source: UCI Machine Learning Repository
- Type: Bank marketing dataset

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Data Cleaning)

## Analysis Performed
- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Customer segmentation analysis
- Trend and pattern identification

## Key Insights
- Customers without loans showed higher subscription rates
- Certain age groups responded more positively to campaigns
- Job category influenced likelihood of subscription
- Contact method affected customer engagement

## Business Recommendations
- Target high-performing customer segments
- Focus marketing on responsive demographics
- Optimize communication channels
- Improve campaign targeting strategies

## Project Structure
- `data/` → raw dataset
- `analysis/` → Excel analysis file
- `dashboard/` → dashboard visuals
